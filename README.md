# ConvoNet

A real-time chat application built with Django.

## About

ConvoNet is a Django-based web application that enables real-time chat functionality. Users can create chat rooms, send direct messages, and communicate in real-time.

## Features

- Real-time messaging
- Chat rooms
- Direct messaging (DM)
- Media file sharing
- User-friendly dashboard

## Installation

### Prerequisites
- Python 3.x
- Django

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/ketanjain113/ConvoNet.git
   cd ConvoNet/chatroom
   ```

2. Install dependencies:
   ```bash
   pip install django
   # Install additional dependencies as needed
   # For production, check if there's a requirements.txt file:
   # pip install -r requirements.txt
   ```

3. Run migrations:
   ```bash
   python manage.py migrate
   ```

4. Start the development server:
   ```bash
   python manage.py runserver
   ```

5. Open your browser and navigate to `http://127.0.0.1:8000/`
